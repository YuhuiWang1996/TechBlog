### 不同级别
1. **类型选择器** : (h1 ul li ...) ; **伪元素** : (::before)
2. **类选择器**，**属性选择器**和**伪类**
3. ID选择器

> 通配选择器(*)，关系选择器(+,>,~,' ',||)和否定伪类(:not())对优先级没影响

- 当在一个样式声明中使用一个 ==!important== 规则时，此声明将**覆盖任何其他声明**。
- 给元素添加的**内联样式**总会覆盖外部样式表的任何样式 ，因此可看作是具有**最高的优先级**。

## 同一级别
1. 后写的会覆盖先写的样式。
2. css引入方式不同，优先级不同。
	1. **内联（行内）样式** (超越同级别)
	2. 内部样式表
	3. 外部样式表
	4. 导入样式(@import)
